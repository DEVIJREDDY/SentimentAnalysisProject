\# Sentiment Analysis Project



\## Overview

This project performs sentiment analysis using machine learning techniques.



\## Technologies Used

\- Python

\- Pandas

\- Scikit-learn

\- Matplotlib

\- Jupyter Notebook



\## Features

\- Text preprocessing

\- TF-IDF vectorization

\- Logistic Regression model

\- Sentiment prediction

\- Accuracy evaluation

\- Confusion matrix visualization



\## Dataset

The dataset contains positive and negative text reviews.



\## How to Run



Install libraries:



pip install pandas scikit-learn matplotlib notebook



Run notebook:



jupyter notebook



\## Output

\- Accuracy score

\- Classification report

\- Confusion matrix graph

\- Sentiment prediction
